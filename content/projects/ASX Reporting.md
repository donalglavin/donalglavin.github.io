# Introduction.
Analysing the performance of the Australian Securities Exchange (ASX) offers valuable insights into the nation's key
economic drivers and current investment trends.

To track these market movements efficiently, I developed an automated pipeline utilizing GitHub Actions to generate and
publish a concise weekly overview of the ASX.

You can view the latest updates at [The ASX Report](https://donalglavin.github.io/asxReporting). To optimize CI/CD
resources and minimize GitHub Actions execution minutes, this reporting tool is maintained in its own dedicated
repository.
